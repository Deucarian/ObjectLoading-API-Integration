# Support for Deucarian Object Loading API Integration

Use [GitHub Issues](https://github.com/Deucarian/ObjectLoading-API-Integration/issues) for reproducible defects at the API/Object Loading adapter boundary, documentation gaps, and narrowly scoped feature requests. Search existing issues before opening a new one.

Include all three package versions, the Unity editor version and target platform, the smallest API response and loading sequence that reproduces the behavior, expected and actual results, and diagnostics with endpoints, credentials, and signed URLs removed.

`main` is the stable package channel. `develop` is a preview channel and may change before promotion. Please reproduce against the applicable current channel before reporting an older snapshot.

Do not disclose vulnerabilities, tokens, private endpoints, signed URLs, or user data in an issue. Follow [SECURITY.md](SECURITY.md) for private security reports.

Support is provided on a best-effort basis. Defects isolated to API or Object Loading should be reported in the owning package repository.
